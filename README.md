# Space Invaders Emulator

1978 space invaders emulator.  
`i8080.odin` also emulates all of intel's 8080 instructions.

## Key mappings

```
move left: A, <-
move right: D, ->
insert coin: 3, return
fire: space
1 player start: 1
2 players start: 2
```

## Build

### Windows

``bash
./build.bat #release or debug
``

### Linux

``bash
./build.sh #release or debug
``
